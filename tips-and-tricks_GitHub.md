## Getting Started

### New GitHub Repository

- Web
    1. Click on Repositories tab
    2. Click on New

- Local
1. Create a project
2. Navigate to the root folder
3. Run the command
```sh
git init
git add .
git commit -m 'first commit'
git branch -M main
git remote add origin https://github.com/username/example-project.git
git push -u origin main
```

### Commands

- Push
```sh
git push -u origin main
```

- Pull
```sh
git pull
```
